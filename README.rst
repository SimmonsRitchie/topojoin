========

A lightweight utility to left join topojson data with CSV data.

* Free software: MIT
* Documentation: https://topojoin.readthedocs.io.

Install
----------

::

    pip install topojoin


Basic usage
-----------

Command line
============

::

   topojoin ny

   >> Initializing getpop
   >> Selected states: ['ny']
   >> Processing: New York
   >> getpop complete
